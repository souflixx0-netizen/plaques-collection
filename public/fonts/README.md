# Polices personnalisées

Placez ici les fichiers des deux polices nécessaires :

## UKNumberPlate
- Fichiers : `UKNumberPlate.woff2` + `UKNumberPlate.woff`
- Source : https://www.dafont.com/uk-number-plate.font (licence gratuite pour usage personnel/commercial)
- Alternative : https://fonts.cdnfonts.com/css/uk-number-plate

## Gunplay RG
- Fichiers : `GunplayRg.woff2` + `GunplayRg.woff`
- Source : https://www.dafont.com/gunplay.font (licence Ray Larabie — gratuite)
- Alternative : rechercher "Gunplay Regular" sur dafont.com

## Conversion en WOFF2
Si vous avez le .ttf ou .otf, convertissez via :
https://cloudconvert.com/ttf-to-woff2

Les autres polices (Stencil, American Typewriter, Calibri Bold)
sont des polices système et ne nécessitent pas de fichiers supplémentaires.
